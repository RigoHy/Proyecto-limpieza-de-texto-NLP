# Haciendo leer tu libro favorito a tu computadora

## Descripción

Proyecto de procesamiento de lenguaje natural donde se realiza limpieza de texto a un libro en formato `.txt`.

## Procesos realizados

- Normalización
- Conversión a minúsculas
- Eliminación de números
- Eliminación de caracteres especiales
- Lematización

## Tecnologías utilizadas

- Python
- NLTK
- spaCy

## Instalación

```bash
pip install -r requirements.txt
